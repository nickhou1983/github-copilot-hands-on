## GitHub Copilot Lab

### 什么是 Agent Skills？

Agent Skills 本质是包含指令、脚本和资源的文件夹，GitHub Copilot 可以在相关任务时加载这些技能来执行专门的任务。它是一个**开放标准**，可跨多个 AI 代理工作，包括：

- **GitHub Copilot in VS Code**
- **GitHub Copilot CLI**
- **GitHub Copilot coding agent**

核心特点：
- **模块化**：每个 Skill 是一个独立的文件夹，包含 SKILL.md 说明文件和相关脚本/资源
- **可复用**：同一技能可以在不同项目和工作区中使用
- **社区共享**：可以使用社区提供的现成技能，也可以创建自己的技能
- **自动触发**：Copilot 会根据任务内容自动匹配并加载相关技能

了解更多关于 Agent Skills 的内容：https://agentskills.io/home

### 在本 Lab 中的应用

在本实验中，我们将：

- 了解如何启用和配置 Agent Skills
- 使用社区提供的 Agent Skill（以 pptx 技能为例）
- 体验 Skill 的自动加载和执行流程

## Lab 步骤

### 第一步：启用 Agent Skills 支持

#### 1.1 目标

在 VS Code 中启用 Agent Skills 功能

#### 1.2 操作步骤

1. **打开 VS Code 设置**

   VS Code 默认没有启用 Agent Skills 支持，需手动启用。打开 VS Code 设置（`Cmd + ,`），搜索 `agent skills`

2. **启用配置**

   找到 `Chat > Agent: Skills` 相关设置，勾选启用

3. **了解技能存储位置**

   GitHub 支持将 Skills 文件夹存储在两个位置：

   - **个人技能目录**：`~/.copilot/skills/`（适用于所有项目）
   - **项目技能目录**：`<项目根目录>/.github/skills/`（仅适用于当前项目）

#### 1.3 验证

- VS Code 设置中 Agent Skills 已启用

---

### 第二步：安装社区提供的 Agent Skill

#### 2.1 目标

从社区下载并安装一个 Agent Skill

#### 2.2 操作步骤

1. **浏览社区技能**

   访问社区技能仓库查看可用技能：https://github.com/anthropics/skills/tree/main/skills

2. **下载 pptx 技能**

   以 pptx 技能为例，将其下载到个人技能目录：

   ```bash
   mkdir -p ~/.copilot/skills
   cd ~/.copilot/skills
   ```

   将 pptx 技能文件夹复制到该目录下。完成后目录结构如下：

   ```
   ~/.copilot/skills/
   └── pptx/
       ├── SKILL.md
       └── ...
   ```

3. **查看已安装的技能**

   在 VS Code 中打开 Copilot Chat，选择 Agent 模式，输入以下提示词：

   ```
   你当前支持哪些skills？
   ```

   Copilot 会列出当前可用的所有技能

#### 2.3 验证

- pptx 技能文件夹已存在于 `~/.copilot/skills/` 目录
- Copilot Chat 能够识别并列出已安装的技能

---

### 第三步：使用 Agent Skill 执行任务

#### 3.1 目标

使用已安装的 pptx 技能生成 PPT 文件

#### 3.2 操作步骤

1. **使用 pptx skill 生成 PPT**

   在 VS Code 中打开 Copilot Chat，选择 Agent 模式，输入以下提示词：

   ```
   生成一个宣传当前项目的 PPT，包含以下内容，包括项目简介、核心功能、技术架构和未来规划。
   ```

2. **观察执行过程**

   注意观察 Copilot 的执行流程：
   - Copilot 自动识别任务与 pptx 技能相关
   - 加载 SKILL.md 中的指令
   - 按照技能定义的流程执行任务

3. **查看生成结果**

   确认 PPT 文件已生成，并检查内容是否符合要求

#### 3.3 验证

- PPT 文件已成功生成
- 内容结构完整，包含指定的页数

---

### 第四步：了解 Skill 的结构（可选）

#### 4.1 目标

了解 Agent Skill 的文件结构，为后续创建自定义 Skill 做准备

#### 4.2 操作步骤

1. **查看 SKILL.md 文件**

   打开已安装的技能文件夹，查看 SKILL.md 的结构：

   ```bash
   cat ~/.copilot/skills/pptx/SKILL.md
   ```

   一个典型的 SKILL.md 包含：
   - **YAML 头部**：定义技能名称、描述和触发条件
   - **指令正文**：详细的执行步骤和规则
   - **示例**：使用该技能的提示词示例

2. **理解触发机制**

   Agent Skills 的触发依赖于 SKILL.md 中的 `description` 字段。当用户的提示词与描述匹配时，Copilot 会自动加载该技能

#### 4.3 验证

- 理解 SKILL.md 的基本结构
- 了解技能的自动触发机制
