## GitHub Copilot Lab

### What are Agent Skills?

Agent Skills are essentially folders containing instructions, scripts, and resources that GitHub Copilot can load to perform specialized tasks. It is an **open standard** that works across multiple AI agents, including:

- **GitHub Copilot in VS Code**
- **GitHub Copilot CLI**
- **GitHub Copilot coding agent**

Core Features:
- **Modular**: Each Skill is an independent folder containing a SKILL.md description file and related scripts/resources
- **Reusable**: The same skill can be used across different projects and workspaces
- **Community Sharing**: You can use ready-made skills from the community or create your own
- **Auto-triggered**: Copilot automatically matches and loads relevant skills based on task content

Learn more about Agent Skills: https://agentskills.io/home

### Application in This Lab

In this lab, we will:

- Learn how to enable and configure Agent Skills
- Use a community-provided Agent Skill (using pptx skill as an example)
- Experience the automatic loading and execution flow of Skills

## Lab Steps

### Step 1: Enable Agent Skills Support

#### 1.1 Objective

Enable the Agent Skills feature in VS Code

#### 1.2 Operation Steps

1. **Open VS Code Settings**

   VS Code does not enable Agent Skills by default. You need to enable it manually. Open VS Code settings (`Cmd + ,`), search for `agent skills`

2. **Enable the Setting**

   Find the `Chat > Agent: Skills` related setting and check to enable it

3. **Understand Skill Storage Locations**

   GitHub supports storing Skills folders in two locations:

   - **Personal skills directory**: `~/.copilot/skills/` (applies to all projects)
   - **Project skills directory**: `<project root>/.github/skills/` (applies to current project only)

#### 1.3 Verification

- Agent Skills is enabled in VS Code settings

---

### Step 2: Install a Community Agent Skill

#### 2.1 Objective

Download and install an Agent Skill from the community

#### 2.2 Operation Steps

1. **Browse Community Skills**

   Visit the community skills repository to see available skills: https://github.com/anthropics/skills/tree/main/skills

2. **Download the pptx Skill**

   Using the pptx skill as an example, download it to your personal skills directory:

   ```bash
   mkdir -p ~/.copilot/skills
   cd ~/.copilot/skills
   ```

   Copy the pptx skill folder to this directory. The resulting directory structure should be:

   ```
   ~/.copilot/skills/
   └── pptx/
       ├── SKILL.md
       └── ...
   ```

3. **Check Installed Skills**

   Open Copilot Chat in VS Code, select Agent mode, and enter the following prompt:

   ```
   What skills do you currently support?
   ```

   Copilot will list all currently available skills

#### 2.3 Verification

- The pptx skill folder exists in `~/.copilot/skills/` directory
- Copilot Chat can recognize and list the installed skills

---

### Step 3: Use an Agent Skill to Execute a Task

#### 3.1 Objective

Use the installed pptx skill to generate a PPT file

#### 3.2 Operation Steps

1. **Use the pptx Skill to Generate a PPT**

   Open Copilot Chat in VS Code, select Agent mode, and enter the following prompt:

   ```
   Use the pptx skill to create a PPT about "GitHub Copilot Feature Overview" with 5 slides
   ```

2. **Observe the Execution Process**

   Pay attention to Copilot's execution flow:
   - Copilot automatically recognizes the task is related to the pptx skill
   - Loads instructions from SKILL.md
   - Executes the task following the workflow defined in the skill

3. **Review the Generated Result**

   Confirm the PPT file has been generated and check if the content meets requirements

#### 3.3 Verification

- PPT file has been successfully generated
- Content structure is complete with the specified number of slides

### Step 4: Understand Skill Structure (Optional)

#### 4.1 Objective

Understand the file structure of an Agent Skill to prepare for creating custom Skills later

#### 4.2 Operation Steps

1. **View the SKILL.md File**

   Open the installed skill folder and examine the SKILL.md structure:

   ```bash
   cat ~/.copilot/skills/pptx/SKILL.md
   ```

   A typical SKILL.md contains:
   - **YAML header**: Defines skill name, description, and trigger conditions
   - **Instruction body**: Detailed execution steps and rules
   - **Examples**: Prompt examples for using the skill

2. **Understand the Trigger Mechanism**

   Agent Skills triggering relies on the `description` field in SKILL.md. When the user's prompt matches the description, Copilot automatically loads the skill

#### 4.3 Verification

- Understand the basic structure of SKILL.md
- Understand the automatic trigger mechanism of skills
