Agent Skills

通过技能文件扩展 Copilot 的专门能力

课程定位

介绍 Agent Skills 的概念、启用方式和技能安装流程，并用社区 pptx 技能作为案例。

建议讲解提纲

建议按以下 3 个重点展开讲解，并结合现场演示说明价值。

  • Skill 本质是带有说明、脚本和资源的能力包
  • 支持放在个人目录或项目目录，便于共享和复用
  • 通过实际安装技能，体验自动匹配与调用的工作流

参考文件

github-copilot-lab/lab/cn/03.AgentSkill.md

讲师提示

该文件重点说明 Copilot 如何通过模块化技能扩展任务处理范围。

演示建议

  1. 展示一个技能目录结构，说明 SKILL.md 与脚本资源的关系。
  2. 再演示安装或触发一次技能，让学员看到自动调用效果。
